## 🧑‍💼 Experience

**Data Engineering Intern — [Amadeus](https://www.amadeus.com)** *(Feb 2024 – Aug 2024)*
- Integrated competitor pricing signals from 3 data sources into the ML Boosted Forecast (MLBF) pipeline for airline Revenue Management
- Built a PySpark pipeline on Databricks reducing schema from 150 → 30 columns across 700+ partitions; engineered features across 7 time windows (60–1440 min before departure)
- Used SHAP analysis to identify highest-impact features and introduced a competition-pressure correction term to the forecast model
- **34.3% RMSE improvement** on departure-date forecasts · **12.9%** on days-before-departure · estimated **~$120M revenue uplift** for a $30B airline
- Stack: `Python` · `PySpark` · `Databricks` · `Azure` · `MongoDB` · `SHAP` · `Feature Engineering`

**Machine Learning Research Intern — [Inria](https://www.inria.fr)** *(Feb 2023 – Jul 2023)*
- Researched transformers and variational autoencoders (VAEs) for deep representation learning
- Built a VAE in PyTorch to reduce data dimensionality for dynamic system analysis
- Stack: `Python` · `PyTorch` · `VAEs` · `Transformers`

**Software Engineering Intern — Laboratoire d'Informatique de Grenoble** *(Feb 2022 – Jul 2022)*
- Implemented DSL-to-automated-planning translation using a meta-model for Petri nets
- Stack: `DSL` · `Modelling` · `Automated Planning`
